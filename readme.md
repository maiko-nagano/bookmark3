# プロダクト名
ブックマークアプリ（Session・ログイン）

## 課題内容（どんな作品か）

- SESSIONを使ってログインするとブックマーク一覧が表示される
- ほかのページ（ブックマーク登録、ブックマーク編集、ユーザー登録、ユーザー一覧）もログイン時のみ入れる
- ログアウトするとみられなくなる
